# Distance-measurement-using-ultrasonic-sensors - group project

#Project Description
Build a distance measurement system using ultrasonic waves with an HC-SR04 sensor. Achieve high accuracy in distance calculation by precisely capturing reflected signals through the peripherals of an 8051 microcontroller and processing those signals. The measured distance will be displayed on an LCD screen.
